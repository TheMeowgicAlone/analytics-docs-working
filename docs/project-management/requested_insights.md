## How to Promote to a Request
- Confirm the problem, audience, and desired phase (minimum/operational/acceptable/exceptional).
- Fill out the [Dashboard Feature Request](../templates-and-checklists/dashboard-feature-request.md) template with metrics, data sources, and acceptance criteria.
- Link the completed request here and in the working channel for triage.
